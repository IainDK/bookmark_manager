# bookmark_manager

User Stories
----
```
As a User
So that I have quicker access to the sites I visit regularly
I want to see the list of my bookmarks
```

```
As a User
So that I can save the link to my bookmarks
I want to be able to add a new link with a title to my list
```

```
As a User
So that I can quickly find web sites I bookmarked recently
I would like to see the list in descending chronological order
```


```
As a User
So that I could organise the links for a quicker search
I want to be able to add tags to my links
```

```
As a User
So that I could search my list quicker
I want to filter the list by the tag
```
